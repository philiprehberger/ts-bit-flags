# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `defineFlags()` for creating type-safe flag sets
- `has()`, `set()`, `unset()`, `toggle()` bitwise operations
- `toArray()` and `fromArray()` conversions
- `all` and `none` constants
